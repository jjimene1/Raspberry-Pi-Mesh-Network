# Raspberry-Pi-Mesh-Network

The following is a guide to setting up a functioning Raspberry Pi Mesh Network over Wifi interfaces or ethernet interfaces as well as its application for IoT devices. The code necessary to make the setup can be found in this paper [here](Raspberry_Pi_Mesh_Network-1.pdf).If you are interested in IoT applications and creating a react app more documentation can be found in this [link](website/README.md).
